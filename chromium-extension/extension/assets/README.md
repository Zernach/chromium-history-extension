# Extension Icons

This directory should contain the extension icons in the following sizes:
- icon-16.png (16x16 pixels)
- icon-48.png (48x48 pixels)
- icon-128.png (128x128 pixels)

## Temporary Solution

For development purposes, you can use any placeholder images or generate simple icons using online tools like:
- https://www.favicon-generator.org/
- https://www.canva.com/

## Design Recommendations

The icon should represent:
- Browser history (clock/timeline icon)
- Chat/conversation (speech bubble)
- AI/intelligence (brain or sparkle)

Suggested colors: Blue (#4a90e2) and white

## Creating Icons

You can use tools like:
- Figma
- Adobe Illustrator
- Inkscape (free)
- GIMP (free)

Or use AI image generators to create simple, modern icons.
