History Chat Settings

AI Access Configuration

Choose how to process your browsing history: use the managed backend (no API key required) or provide your own OpenAI API key for direct access.

Two Modes Available:
  • Backend Mode (Default): Leave API key empty. Your browsing history is sent to the managed backend for processing.
  • Direct OpenAI Mode: Provide your own API key. Calls OpenAI directly from the extension with your key.

Chromium History Chat Extension

This extension allows you to chat with OpenAI about your browsing history. Data is now transmitted to the backend for processing and filtering before reaching OpenAI.

Privacy Note:
Backend Mode: Browsing history is sent to the backend service, which then forwards relevant context to OpenAI.
Direct OpenAI Mode: Browsing history is sent directly to OpenAI with your API key. No data goes through the backend.
